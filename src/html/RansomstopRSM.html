중앙관리콘솔 RSM

중앙관리콘솔을 통해 맨드포인트 보안체계를 언제 어디서나 효율적이고 쉽게 통합 관리할 수 있습니다. RansomStop 맨드 포인트 에이전트를 RSM 콘솔을 통하여 관리 및 설정할 수 있도록 제공됩니다. 콘솔을 사용하면 라이선스를 관리 할 수 있으며 맨드 포인트에서 생성 된 모든 위협 및 이벤트에 대한 중앙 관리 대시보드를 통해 시각적으로 제공합니다.

  • 효율적인 보안 통합 관리

    조직내 RansomStop 에이전트들의 보안 상태와 현황을 한 눈에 보고，보안이슈를 해결하며 웹브라우저 인터페이스를 제공하여 외부에서도 조치를 취할수 있도록 지원합니다.

  • 보안정책 관리

    RansomStop 에이전트를 통해 조직 및 사용자 보안정책을 설정하고 중앙 통제 관리할 수 있습니다. 실시간 자동 적용되어 원격지에 있어도 더 쉽고 안전하게 운영할 수 있습니다.

  • 웹 기반 콘솔과 직관적 대시보드

    HTML5로 구성된 웹 기반 콘솔에서 동작하여 웹브라우저를 통한 보안상태를 현황을 한눈에 파악이 가능한 직관적인 대시보드와 시각화된 GUI를 제공합니다.

  • 보안 리포팅

    사용자 현황，악성코드, IT 자산，패치 상태，취약점 현황 등 사내 각 종 보안정보 관련 보고서 제공 상급자에게 즉시 보고가 가능한 형태의 보고서 제공합니다.

주요 기능

기업형 고객에게 중앙 보안 관리 콘솔 RSM은 다양한 기능을 지원합니다.

  • 대시보드

    RansomStop 에이전트를 통합 관리 할 수 있으며, 엔드포인트 보안 현황를 중앙에서 빠르고 직관적으로 확인이 가능하도록 지원합니다

  • 보안정책 관리

    관리자는 보안정책 그룹을 정의하고 상황별 요구사항에 따라 유연한 정책을 설정할 수 있습니다.

  • 에이전트 및 그룹 관리

    관리자는 에이전트 라이선스를 통합관리 할 수 있으며, 또한 기관 조직을 기반으로 에이전트를 관리할 수 있습니다.

  • 업데이트 관리

    RSM을 통하여 에이전트 설치파일을 일괄 배포/업데이트하여 상시 최신 상태를 유지합니다.

  • 보안로그 관리

    에이전트를 통해 수집된 보안로그를 효과적인 관리 및 모니터링 할 수 있도록 관리합니다.

  • 프로세서 관리

    조직내 에이전트에게 수집된 보안로그를 분석하여 보안 위험을 관리자에게 리포트형태로 제공합니다.

  • OS
    • Windows Server 2003 SP2 이상 / 2008 / 2012 / 2016(R2포함)

    • CentOS 6.X이상
  • CPU
    • 최소 : 1GHz 32 비트 (x86) 또는 64 비트 (x64) Intel Pentium
    • 권장 : 2GHz 32 비트 (x86) 또는 64 비트 (x64) Intel Pentium 이상
  • MEMORY
    • 최소 : 4GB 이상
    • 권장 : 8GB 이상
  • HDD
    • 최소 : 50GB 이상의 여유 디스코 공간
    • 권장 : 100GB 이상의 여유 디스코 공간

    • MariaDB(내장) : Windows XP SP3 이상
