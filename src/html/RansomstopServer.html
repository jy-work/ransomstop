서버용 안티랜섬웨어 RansomStop

RansomStop서버는 강력한 서버 보안 기능을 통해 기업 정보 자산을 보호하고 안정적인 서버운영을 지원합니다.
다양한 서버 환경에 맞게 사전 방역 및 다중 보안기능을 통해 랜섬웨어 공격을 차단해 주는 서버 전용 엔드포인트 보안 제품입니다.

  • LWS 기반의 휴리스틱 탐지 기술

    빠르고 가벼운 LWS(Light Weight Signature)기반의 강력한 휴리스틱 알고리즘은 신종 및 변종 랜섬웨어 공력으로 부터 효과적이고 지속적으로 서버내 정보 자산을 보호하여 드립니다.

  • 차단/격리 및 파일 백업

    랜섬웨어의 공격 행위 추적과 해당 행위의 프로세스의 무결성을 검증하여 악의적 공격 프로세서를 사전 차단/격리하고 프로세서가 직접 행위를 시도한 파일에 대해서는 훼손 직전에 안전한 백업을 진행합니다.

  • 효율적인 중앙 관리

    RSM과의 연동을 통해 서버 위험을 즉각적으로 확인이 가능하고 처리학 가능하며, 에이전트 관리, 에이전트 업데이트, 개별/그룹별 보안정책 적용을 다양한 운영관리 기능을 지원합니다

주요 기능 및 특장점

안정적으로 정보자산 보호와 운용 편의성을 위한 다양한 기능을 제공합니다.

  • LWS(Light Weight Signature)

    주요 프로세스에 대해 시그니처를 대조하여 쉽고 빠르게 탐지 및 진단하는 기술이며，서버 사용성 저하없이 빠르게 탐지 및 방어가 가능합니다.

  • 휴리스틱(Heuristics) 보안

    실시간 행위 인식을 통한 휴리스틱 보안 방식은 시그니처 데이터 의존도가 높은 백신 프로그램으로 탐지하기 어려운 신·변종 랜섬웨어도 탐지 및 차단합니다.

  • 데이터 백업/복원

    실시간 행위 인식을 통한 휴리스틱 보안 방식은 시그니처 데이터 의존도가 높은 백신 프로그램으로 탐지하기 어려운 신·변종 랜섬웨어도 탐지 및 차단합니다.

  • MBR(Master Boot Record) 보호

    실시간 행위 인식을 통한 휴리스틱 보안 방식은 시그니처 데이터 의존도가 높은 백신 프로그램으로 탐지하기 어려운 신·변종 랜섬웨어도 탐지 및 차단합니다.

  • 스케줄 백업 기능

    사용자의 요청에 따라 수동으로 스케줄을 조정하여 백업을 할 수 있는 기능
    (파일확장자， 파일명 등 지정하여 해당 파일만 백업하는 기능)

  • 프로세서 예외처리

    사용자의 요청에 의해 프로그램을 해당 프로세스를 예외 처리 하는 기능(차단 또는 허용).

  • 중앙관리서버 연동

    여러 서버 에이전트 위협관리 및 보안운영이 가능한 RSM 연동 프로토콜 을 제공하여 효율적인 보안 유지 관리를 지원합니다. .

  • 운영 안정성

    서버전용 어플라인언스 S/W 제공과 MBR 보호를 통해 운영 안정성을 제공합니다.

사용 환경

RansomStop Server 설치를 위해 아래와 같은 사용 환경을 필요로 합니다.

  • OS
    • Microsoft Windows Server 2019 (64 비트)
    • Microsoft Windows Server 2016 (64 비트)
    • Microsoft Windows Server 2012 R2 Standard / Datacenter (64 비트)
    • Microsoft Windows MultiPoint Server 2012 Standard (64 비트)
    • Microsoft Windows Server 2012 Standard / Essentials / Foundation / Storage Server / Datacenter (64 비트)
    • Microsoft Windows SBS 2011 Standard / Essentials
    • Microsoft Windows 2008 Server R2 Web / Standard / Enterprise / Datacenter (64 비트)
    • Microsoft Windows 2008 Server Web / Standard / Enterprise (32 비트 / 64 비트) / Datacenter (64 비트)
    • Microsoft Windows Server 2003 R2 웹 / Standard / Enterprise / Datacenter
    • Microsoft Windows Server 2003 Web / Standard / Enterprise (32 비트 / 64 비트)

    [주의사항] Windows 2016 Server 및 Windows 2019 Server의 경우 RansomStop Server를 설치하기 전에 Windows Defender를 제거하십시오.
    콜라이언트를 Windows 2016 Server 또는 Windows 2019 Server로 업그레이드하는 경우 업그레이드 후 Windows Defender를 제거합니다.

  • CPU
    • 최소 : 1GHz 32 비트 (x86) 또는 64 비트 (x64) Intel Pentium
    • 권장 : 2GHz 32 비트 (x86) 또는 64 비트 (x64) Intel Pentium 이상
  • MEMORY
    • 최소 : 2GB
    • 권장 : 4GB 이상
  • HDD
    • 최소 : 4,800MB의 여유 디스크 공간
    • 권장 : 10,000MB의 여유 디스크 공간
